# Password Reset and MFA Issues

## Summary

Customers may need help with password resets, verification codes, or multi-factor authentication. Support agents must protect account security and verify identity before making account-level changes.

## Standard Handling

For password reset issues, confirm the login email and ask the customer to check spam folders and corporate email filters. For verification code issues, confirm whether the customer can receive other system emails.

For MFA issues, agents should collect the account email, error message, device status, and whether the customer still has access to backup codes or an admin user.

## Customer-Safe Wording

"I can help troubleshoot the login issue. For security, I may need to verify account ownership before making any changes to authentication settings."

## Escalation

Escalate if the customer requests MFA removal, admin transfer, account ownership change, or access to an account they cannot verify.
